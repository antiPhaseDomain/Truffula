JSON Debugger <json-debug>

The full JSON object can be printed to the screen with the <json-debug></json-debug> tag. This example uses Swagger's example swagger.json.

Example

<json-debug></json-debug>

Root Level Objects <json-key>

The top layer can be accessed by using the json's key as an element: <json-key></json-key>.

Example

<json-host><json-host>

Nested Objects <json-key>

If the root level value is another object, nested objects can be used: 

<json-key>
    <key-childKey></key-childKey>
</json-key>

Example

<json-info>
    <info-title></info-title>
</json-info>
